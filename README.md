## 1. 스마트팜 기반 도시 정신건강 솔루션

> 석사과정 | 2025 데이터안심구역 활용 공동 경진대회 **🏆 우수상**

### Overview

2030 직장인의 우울감 감소를 목표로, 도시 내 스마트팜 최적 입지를 선정하고 이를 실제 서비스로 연결한 프로젝트입니다.
LG U+에서 제공하는 카드·통신 데이터를 데이터안심구역에서 활용하여 타겟층의 유동성과 소비 분포를 파악하고, 공간통계로 최적 입지를 도출했습니다.
입지 선정 이후에는 YOLO 객체탐지와 LLM을 결합한 반려식물 관리 서비스까지 구현하여 분석과 서비스를 하나의 흐름으로 완성했습니다.

### Part 1. 스마트팜 입지 분석

#### Data & Methods

- **데이터**: LG U+ 카드·통신 데이터(데이터안심구역), 유휴공간 데이터, 서울시 공공데이터
- **공간 재구성**: 유휴공간 주소 → 위경도 변환 → 행정동 단위 → 격자(Grid) 단위 재할당
- **공간 보간**: IDW, Kriging, Spline을 활용한 격자 단위 결측 보완
- **공간 자기상관**: Moran's I 전역 검정 + LISA 기반 국지적 핫스팟 도출
- **타겟 분석**: 카드 데이터로 2030 소비 분포 파악, 통신 데이터로 유동 인구 추정

#### Index Design

| 지수 | 구성 요소 |
|------|-----------|
| 치유 잠재력 지수 | 1인가구 비율, 우울 통계, 배달 매출 |
| 접근성 지수 | 대중교통 접근성, 따릉이 대여소, 경사도 |
| 경제 지속성 지수 | 관광·상권 매출 데이터 |

#### Key Contributions

- 행정동 → 격자 단위 공간 재구성으로 분석 해상도 향상
- 공간 자기상관 검정을 통해 단순 분포가 아닌 군집 패턴 기반 입지 도출
- 가중치 시나리오별 입지 유형화로 정책 활용 유연성 확보
- 카드·통신 데이터 결합을 통한 타겟층 소비·유동 패턴 정량화

### Part 2. YOLO 기반 반려식물 관리 서비스

#### Overview

입지 분석 결과와 연계하여, 스마트팜에서 제공할 **디지털 반려식물 관리 서비스 '마음이'** 를 구현했습니다.
YOLO로 식물 상태를 자동 탐지하고, LLM이 진단 결과를 사용자 친화적인 언어로 풀어서 홈페이지로 제공하는 파이프라인입니다.

#### Tech Stack & Pipeline

```
[입력: 이미지 / 영상]
        ↓
[YOLO Object Detection]
 - 식물 및 이상 부위 탐지
 - Bounding Box 좌표 추출 및 후처리
        ↓
[LLM 진단 설명 생성]
 - 탐지 결과를 자연어로 변환
 - 질병 상태·관리 방법 안내
        ↓
[웹 서비스 제공]
```

- **Python**, **YOLO**, **OpenCV**, **LLM API**

#### Key Contributions

- YOLO 모델 추론 파이프라인 구축 및 Bounding Box 후처리 로직 설계
- LLM 프롬프트 설계로 기술적 탐지 결과를 사용자 친화적 설명으로 변환
- 입지 분석(공간통계) → 서비스 구현(CV + NLP)까지 end-to-end 프로젝트 수행

### Skills

`Python` `Spatial Statistics` `GeoPandas` `PySAL` `Moran's I` `LISA` `Kriging / IDW` `YOLO` `OpenCV` `LLM` `LG U+ 카드·통신 데이터`

## 2. 강서구 골목상권 활성화 방안

> 석사과정 | 강서구 빅데이터 분석 아이디어 공모전 **🥉 장려상**

### Overview

매출의 시간적 변동성이 높은 상권을 **불안정 상권**으로 정의하고, 공간통계를 활용해 이를 식별한 뒤, 강서사랑상품권의 차등 할인율 정책으로 활성화하는 방안을 제안한 프로젝트입니다.

### Data & Methods

- **데이터**: 서울시 상권분석서비스 데이터 (행정동 단위 업종별 매출)
- **시계열 군집**: 행정동별 소비 시계열 정규화 → DTW 거리 계산 → Agglomerative Clustering
  - Silhouette Score 기반 최적 군집 수 결정
  - 안정형 / 변동형 상권 유형화
- **영향 변수 도출**: VIF 기반 다중공선성 제거 → 로지스틱 회귀(불안정 상권 여부 예측)
- **공간회귀**: Moran's I로 공간적 자기상관 확인 → Queen 인접행렬 기반 Spatial Lag Model(SAR)
- **정책 대상 선정**: 300m buffer 분석으로 최종 정책 적용 후보 상권 도출

### Key Contributions

- 시계열 변동성 기반 상권 유형화로 정책 우선순위 객관화
- 공간 자기상관을 고려한 SAR 모델 적용으로 OLS 대비 추정 편향 개선
- 공모전 심사 피드백 반영: 타 지역화폐(서울사랑상품권 등) 비교 분석 및 할인율 구체화
- 논문 기반 이론적 근거 + 실증 데이터 분석 + 정책 설계의 통합 구조 구현

### Tech Stack

`Python` `pandas` `numpy` `tslearn (DTW)` `Agglomerative Clustering` `Logistic Regression` `GeoPandas` `PySAL` `Spatial Lag Model` `Moran's I` `matplotlib`

## 3. LG Aimers 수료

### 1. 초거대 언어 모델(LLM)의 효율적 추론을 위한 고도화된 경량화 전략 수행
단순한 모델 압축을 넘어, 모델 성능 저하를 최소화하면서 추론 속도를 극대화하는 것을 목표로 기술적 최적화를 단행. 구체적으로 양자화(Quantization), 지식 증류(Knowledge Distillation), 프루닝(Pruning) 등 다각도의 경량화 기법을 EXAONE 모델에 적용. 특히 FP16/BF16 수준의 정밀도를 4-bit 및 8-bit(INT4/INT8) 수준으로 저감하면서도, 모델 고유의 언어 이해 및 생성 능력을 보존하기 위해 가중치 보정 알고리즘을 정교화.

### 2. 하드웨어 제약 조건을 고려한 On-device AI 최적화 구현
고성능 GPU 자원이 제한적인 실제 비즈니스 환경 및 온디바이스(On-device) 환경을 가정하여 모델의 실용성을 확보. 경량화 모델 설계 시 **Latency(지연 시간)**와 **Throughput(처리량)**을 핵심 지표로 설정하고, 저사양 하드웨어에서도 EXAONE의 고성능 추론이 가능하도록 하드웨어 친화적 알고리즘을 설계. 이를 통해 모델의 경량성과 연산 효율성 사이의 최적의 트레이드오프(Trade-off) 지점을 도출.

### 3. 지식 전이 기법을 통한 도메인 특화 성능 유지 및 고도화
파라미터 축소 과정에서 발생할 수 있는 '정보 손실'을 방지하기 위해, 원본 모델의 복합 사고 능력을 효과적으로 전이시키는 기술을 적용. 제조, 금융 등 특정 산업군에 최적화된 지식을 보존하고자 Fine-tuning과 LoRA(Low-Rank Adaptation) 등 효율적인 파라미터 학습 기법을 활용. 이를 통해 모델 크기는 획기적으로 줄이면서도 도메인 특화 태스크에서의 성능을 안정적으로 유지.

### 4. LLM 특화 평가 메트릭 설계를 통한 정밀한 성능 검증
기존 수치 예측 모델의 한계를 넘어, LLM의 특성을 반영한 다차원 평가 체계를 구축. 단순 수치인 Perplexity(PPL) 외에도 지시 이행 능력(Instruction Following) 및 논리적 추론(Reasoning) 지표를 도입하여 경량화 모델의 품질을 정량적·정성적으로 검증. 이를 바탕으로 경량화된 모델이 원본 대비 생성 답변의 논리적 일관성과 신뢰성을 얼마나 견고하게 유지하고 있는지 증명.
