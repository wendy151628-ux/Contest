## 1. 2025 데이터안심구역 활용 공동 경진대회

서울시 유휴공간 데이터를 활용한 스마트팜 최적 입지 선정과 

YOLO 기반 객체 탐지 서비스를 결합한 도시 정신건강 솔루션 프로젝트입니다.  

2025 데이터안심구역 활용 공동 경진대회에서 **우수상**을 수상했습니다.

### Part 1. 스마트팜 기반 정신건강 도시 솔루션 설계

#### Project Overview
- 2030 세대 우울감 감소를 목표로 한 도시형 스마트팜 입지 분석
- 유휴공간 데이터를 활용한 최적 입지 선정
- 공간 통계 기반 치유 인프라 배치 전략 설계
- 디지털 반려식물 서비스 ‘마음이’ 기획

#### Data & Methods
- 유휴공간 주소 → 위경도 변환
- 행정동 → 격자 단위 재할당
- 공간 보간 (IDW, Kriging, Spline)
- Moran’s I, LISA 기반 Hotspot 분석
- 다중 지표 가중치 설계

#### Index Design
- 치유 잠재력 지수 (1인가구, 우울 통계, 배달매출)
- 접근성 지수 (대중교통, 따릉이, 경사도)
- 경제 지속성 지수 (관광·상권 매출 데이터)

#### Key Contributions
- 격자 기반 공간 재구성 분석 수행
- 공간 자기상관 검정 및 핫스팟 도출
- 가중치 변화에 따른 입지 유형화
- 정책 적용 가능 지역 제안

## 2025 강서구 빅데이터 분석 아이디어 공모전

시간적·공간적 특성을 고려하여 강서구 골목상권 활성화 방안을 설계한 프로젝트입니다.

강서구 빅데이터 분석 아이디어 공모전에서 **장려상(3등상)** 을 수상했습니다.

### Project Overview
- 서울시 상권분석서비스 데이터 활용
- 소비 시계열 기반 상권 유형 분류
- DTW 시계열 군집 분석 수행
- 로지스틱 회귀 기반 영향 변수 도출
- 공간회귀(SAR) 기반 고변동 상권 예측
- 강서사랑상품권 차등 할인 정책 제안

### Tech Stack
- Python: pandas, numpy, matplotlib, statsmodels
- Time Series: DTW (tslearn)
- Clustering: Agglomerative Clustering, Silhouette Score
- Regression: Logistic Regression
- Spatial Analysis: GeoPandas, PySAL, Moran’s I, Spatial Lag Model
- Scaling: MinMax Scaling
- GIS 기반 buffer 분석 및 시각화

### Key Contributions
- 행정동 단위 소비 시계열 정규화 및 DTW 거리 계산
- Silhouette Score 기반 최적 군집 수 도출
- 안정형/변동형 상권 유형화
- VIF 기반 다중공선성 제거 후 로지스틱 회귀 수행
- Moran’s I 검정으로 공간적 자기상관 확인
- Queen 인접행렬 기반 Spatial Lag Model 적용
- 300m buffer 기반 최종 정책 적용 후보 상권 선정

### Achievement
- 강서구 빅데이터 분석 아이디어 공모전 장려상(3등) 수상

### Skills Gained
- 시계열 + 공간 분석 통합 설계 역량
- 공간 자기상관을 고려한 회귀모형 적용 경험
- 공공 데이터 기반 정책 의사결정 구조 설계 능력
- GIS 기반 상권 후보 입지 도출 경험
